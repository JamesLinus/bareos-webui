Bareos WebUI


About

The Bareos WebUI is a free software tool written in PHP, intended to manage the Backup, Archiving and Recovery Software Bareos over the Web.

The WebUI is based on Zend Framework 2 an open source framework for developing web applications and services using PHP 5.3+ and makes extensive use of the Twitter Bootstrap front-end framework.

The Bareos WebUI is licensed under AGPL Version 3.

For more information please visit the new bareos-webui project site.
